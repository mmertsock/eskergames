Keyboard Commands

  • ⬆︎ ⬇︎ ⬅︎ ➡︎: Move map up/down/left/right
  • + −: Zoom in/out
  • ⇧ 1 2 3 4: Change zoom level
  • 1 2 3 4 5 6: Change brush size
  • ESC: Cancel tools, close popups, etc.
  • Q: Dirt brush
  • A: Forest brush
  • W: Freshwater brush
  • S: Saltwater brush
  • R: Build river
  • F: Build forest
  • O: Build ocean
  • L: Build lake
  • X: Destroy a terrain feature
