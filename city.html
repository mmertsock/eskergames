Keyboard Commands

  • ⬆︎ ⬇︎ ⬅︎ ➡︎: Move map up/down/left/right
  • + −: Zoom in/out
  • ⇧ 1 2 3 4: Change zoom level
  • 0: Pause/resume
  • 1 2 3 4: Set game speed
  • ESC: Cancel tools, close popups, etc.
  • ?: Query
  • X: Bulldozer
  • Q W E: Build R/C/I zone
  • T: Plant tree
